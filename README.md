## Description

A Simple boilerplate to kick start your next Nest.js Serverless project.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running Serverless Local
```bash
$ sls offline
```
